FARMTRONICS - 1.0
Add the Farmtronics Computer to your TV, and Farmtronics Bots to your farm.
Both include built-in help and are fully programmable.

:: REQUIREMENTS ::
• Stardew Valley up to date
• Latest version of SMAPI
• Currently supports only desktop

:: FEATURES ::
• Identical programmable computers on TV and on every bot
• 40x20 full-color text display
• Modern, object-oriented language (MiniScript) that is easy to learn
• Bots can use tools, water crops, and plant seeds (given appropriate items of course)
• Get info on the farm or whatever area the bot is in

:: KNOWN ISSUES ::
• Bots may not work outside the farmhouse and farm
• Documentation is still a bit thin
• Currently available only in English

:: GETTING HELP ::
• Please see the Farmtronics Wiki (https://github.com/JoeStrout/Farmtronics/wiki)
for full documentation & sample code
